## 🚀 **QUICK START GUIDE**

### **Prerequisites**
- Node.js 18+ and npm installed ([nvm guide](https://github.com/nvm-sh/nvm#installing-and-updating))
- Git for version control

### **1. Clone & Setup**
```bash
# Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```

### **2. Environment Configuration**
```bash
# Copy environment template
cp .env.example .env

# Add your API keys (check .env for required variables)
# - Supabase credentials (configured in your deployment environment)
# - Financial data API keys
# - AI service credentials
```

### **3. Access Your Application**
- 🌐 **Development:** `http://localhost:5173`
- 🚀 **Production:** Deploy to Vercel, Netlify, or your preferred hosting platform
